# Rhythm Game

A 3D rhythm game built with Three.js and Vite, inspired by Robeats.

## Features

- 4-lane rhythm gameplay
- Notes come from the horizon toward the player
- 3D perspective view with Three.js
- Score and combo system
- Visual hit effects
- Responsive keyboard controls

## Controls

- **D, F, J, K** - Hit notes in respective lanes
- **SPACE** - Start game

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## How to Play

- Press the corresponding key (D, F, J, K) when notes reach the glowing hit zone at the bottom
- Perfect hits (close timing) = 300 points
- Great hits = 200 points
- Good hits = 100 points
- Build combos for higher scores!
